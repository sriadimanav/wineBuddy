🔮 Other useful tools in production setups

🔍 Code Quality / Standards
Stylelint → if you use CSS/SCSS/Tailwind, ensures clean styles.
SonarLint or SonarQube → code smells, maintainability, security.
EditorConfig → keep indentation/line endings consistent across IDEs.

🧪 Testing
Jest / Vitest → unit testing.
Playwright / Cypress → end-to-end testing.
Some teams add pre-push hooks for running a minimal test suite before pushing.

⚡ DX & Automation
Changesets → automated versioning + changelog based on Conventional Commits.
Lint PR titles (GitHub Action using commitlint rules).
Danger.js → automates PR reviews (e.g., “you didn’t update changelog”).
Renovate / Dependabot → keeps dependencies up to date automatically.

🔐 Security
npm audit / pnpm audit / snyk → dependency vulnerability checks.
git-secrets or truffleHog → prevent committing secrets (AWS keys, tokens).

📦 Monorepo/Scaling (if needed)
Nx or Turborepo → caching, task pipelines.
pnpm workspaces → if you’ll have multiple packages.
